VTESPROXI - ARRANQUE EN WINDOWS

1. Requisitos previos

- Tener Git instalado
- Tener Python para Windows instalado
- Durante la instalación de Python, marcar la opción para añadir Python al PATH

2. Archivo a ejecutar

- Haz doble clic en: run_windows_clone.bat

3. Qué hace automáticamente

- Clona el proyecto en tu PC si todavía no existe
- Crea el entorno virtual
- Instala dependencias
- Prepara la base de datos
- La primera vez te pedirá:
  - nombre de usuario
  - clave
- Arranca la app localmente
- Abre el navegador automáticamente

4. Dónde se instala

- %LOCALAPPDATA%\WebVTES\repo

5. Importante

- La primera ejecución puede tardar unos minutos
- En ejecuciones posteriores no vuelve a descargar el repo; simplemente arranca la app
- Si Windows pregunta por permisos de PowerShell o firewall local, acepta para red local

6. Si algo falla

- Cierra la ventana
- Vuelve a ejecutar run_windows_clone.bat
- Si sigue fallando, envíame una captura del error
